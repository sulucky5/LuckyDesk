## 1. 디자인 컨셉

| 항목 | 내용 |
|------|------|
| **테마** | Dark Mode (다크 모드) |

## 2. 디자인 토큰 (CSS Variables)

```css
:root {
    --bg-color: rgba(26, 26, 26, var(--bg-opacity, 0.85));
    --border-color: rgba(255, 255, 255, 0.1);
    --text-main: #FFFFFF;
    --text-dim: #AAAAAA;
    --accent-blue: #3B82F6;
    --accent-red: #FF5F5F;
    --accent-blue-dim: #5F5FFF;
}
```

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--bg-color` | `rgba(26,26,26, 0.85)` | 위젯 배경 (투명도 조절 가능) |
| `--border-color` | `rgba(255,255,255, 0.1)` | 경계선/구분선 |
| `--text-main` | `#FFFFFF` | 주요 텍스트 |
| `--text-dim` | `#AAAAAA` | 보조 텍스트 |
| `--accent-blue` | `#3B82F6` | 액센트 색상, 버튼, 오늘 표시 |
| `--accent-red` | `#FF5F5F` | 일요일, 위험 액션 |
| `--accent-blue-dim` | `#5F5FFF` | 토요일 |

```
     ↕ 8방향 리사이즈 핸들 (편집 모드 시)
```

### 3.2 헤더 영역

```
┌─────────────────────────────────────────────────────────┐
│  ┌────────────┐  ┌──────────────────┐  ┌──────────────┐ │
│  │ 뷰 전환     │  │  네비게이션       │  │ 액션 버튼     │ │
```

| 요소 | ID | 동작 |
|------|-----|------|
| 월간 뷰 버튼 | `view-month` | `dayGridMonth` 뷰로 전환 |

| 설정 버튼 | `settings-btn` | 설정 모달 열기 |
| 닫기 버튼 | `close-btn` | 앱 종료 |

## 5. 뷰 모드별 설계

### 5.1 월간 (Month) 뷰

- FullCalendar `dayGridMonth` 사용
- `expandRows: true` — 화면 높이에 맞춰 행 자동 확장
- 일요일: 빨간색 (`--accent-red`)
- 토요일: 파란색 (`--accent-blue-dim`)
- 오늘: 파란 원형 배경 하이라이트
- 날짜 번호: 좌측 정렬

### 5.2 주간 (Week) 뷰

- FullCalendar `timeGridWeek` 사용
- 0시~24시 타임라인 표시
- `nowIndicator: true` — 현재 시각 표시선

### 5.3 일간 (Day) 뷰

- FullCalendar `timeGridDay` 사용
- 단일 날짜의 상세 타임라인

## 6. 인터랙션 설계

### 6.1 이벤트 인터랙션

| 동작 | 트리거 | 결과 |
|------|--------|------|
| 날짜 더블클릭 | 날짜 셀 더블클릭 | 해당 날짜로 일정 추가 모달 열기 |
| 이벤트 더블클릭 | 이벤트 더블클릭 | 해당 일정 수정 모달 열기 |

> 더블클릭은 300ms 이내 연속 클릭으로 판별

### 6.2 색상 선택

- 7개 프리셋 색상 원형 버튼
- 커스텀 색상 선택기 (`input[type=color]`)
- 선택된 색상에 흰색 테두리 + 글로우 효과

### 6.3 로딩 상태

- 구글 동기화 시 로딩 오버레이 표시
- 완료/에러 시 alert 메시지

| 항목 | 설계 |
